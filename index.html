rofan.ai 대화 추출기

긴 대화 내역을 한 번의 클릭으로 텍스트 파일로 저장합니다.

사용법 (간편 설치)

1
아래에서 원하는 버전의 버튼을 북마크바로 드래그하면 설치 완료!
2
rofan.ai에서 저장하고 싶은 대화 페이지를 열고, 북마크바의 버튼을 클릭합니다.
3
자동으로 모든 대화를 로드한 뒤 .txt 파일이 다운로드됩니다.

참고사항

  • 로그인 필요 — 본인의 대화만 추출할 수 있습니다.
  • 긴 대화는 시간 소요 — 대화가 길수록 로딩에 시간이 걸립니다. 화면을 닫지 말고 기다려주세요.
  • 추출 중 화면에 진행 상황이 표시됩니다.
  • PC 크롬 브라우저에서 사용을 권장합니다.

FAQ

북마크바가 안 보여요

크롬에서 Ctrl+Shift+B (맥: Cmd+Shift+B)를 누르면 북마크바가 나타납니다.

클릭해도 아무 반응이 없어요

rofan.ai 대화 페이지에서만 작동합니다. 대화가 열린 상태에서 클릭해주세요.

중간에 멈춘 것 같아요

대화가 매우 긴 경우 로딩에 수 분이 걸릴 수 있습니다. 화면의 진행 상황 표시를 확인해주세요.

기본 버전과 Clean 버전의 차이는?

Clean 버전은 대화 끝에 붙는 상황 요약 블록을 자동으로 제거합니다. 순수 대화만 원하면 Clean 버전을 사용하세요.
